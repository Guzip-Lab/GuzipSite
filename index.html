Разрабатываемые проекты

  • MizOS

    Linux Kernel Debian Base OS Development

    Собственная операционная система, построенная на базе стабильного ядра Linux и файловой системы (rootfs) Debian.

    Основная цель проекта — глубокое изучение системного администрирования и архитектуры ОС. Вектор развития направлен на постепенный полный отказ от стандартных пакетов Debian и сборку уникальной, независимой custom rootfs с нуля.

  • GuzipSite

    HTML5 CSS3 Minimal GitHub Pages

    Веб-ресурс, выступающий в роли визитной карточки и единой точки входа для всей экосистемы Guzip-Lab.

    Здесь нет ничего лишнего: проект демонстрирует подход к веб-разработке без использования перегруженных JS-фреймворков. Код полностью открыт, оптимизирован для SEO и хостится напрямую через бесплатные серверы GitHub.
